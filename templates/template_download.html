LovexGames 游戏导入模板

使用说明

  1. 点击下方按钮下载CSV模板文件
  2. 使用Excel或其他电子表格软件打开CSV文件
  3. 按照模板格式填写游戏信息
  4. 保存为Excel格式(.xlsx或.xls)
  5. 在后台管理系统中上传填写好的Excel文件
注意：请确保填写的数据格式正确，特别是isFeatured和isNew字段，应填写"true"或"false"（不含引号）。

字段说明

字段名 说明 是否必填 示例
id 游戏唯一标识符，留空则自动生成 否 game-001
title 游戏标题 是 超级马里奥
description 游戏描述 否 一款经典的平台跳跃游戏
embedUrl 游戏嵌入URL 是 https://example.com/games/mario
thumbnailUrl 游戏缩略图URL 是 https://example.com/images/mario.jpg
category 游戏分类 否 动作游戏
rating 游戏评分(0-5) 否 4.5
dateAdded 添加日期，留空则使用当前日期 否 2025年3月
instructions 游戏说明 否 使用方向键移动，空格键跳跃
isFeatured 是否为推荐游戏(true/false) 否 true
isNew 是否为新游戏(true/false) 否 true
